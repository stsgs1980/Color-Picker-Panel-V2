# Color Picker Panel V2

A web application code bundle exported from the Color Picker Panel V2 Figma design project.

[![License: MIT](https://img.shields.io/badge/License-MIT-green?style=flat-square)](LICENSE)

## Tech Stack

- **Design** - Figma
- **Runtime** - Node.js

## Getting Started

### Prerequisites

- Node.js 20+ or Bun

### Installation

```bash
git clone https://github.com/stsgs1980/Color-Picker-Panel-V2.git
cd Color-Picker-Panel-V2
bun install
```

### Run

```bash
bun run dev
```
